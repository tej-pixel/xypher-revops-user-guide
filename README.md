# Xypher RevOps — End-User Guide

An interactive, end-user guide to the Xypher Salesforce RevOps system, organised around the
**bowtie** customer journey (Awareness → Education → Selection → Commit → Onboard → Impact → Grow).

Each stage shows **what the user does** vs **what the system does automatically**, with a live demo
journey (Northwind Logistics) and screenshots from the org.

Built by SaaScend. View: https://tej-pixel.github.io/xypher-revops-user-guide/
